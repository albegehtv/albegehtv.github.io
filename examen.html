I- Redes de Computadoras 5P

Selección Múltiple:

  1. 1) ¿Qué es una dirección IP?

    1. a) Un identificador único para un dispositivo en una red.
    2. b) Un programa para enviar correos electrónicos.
    3. c) El nombre de un sitio web.
    4. d) Un código de barras para routers.

  2. 2) ¿Cuál es la función principal de un router en una red?

    1. a) Conectar dispositivos dentro de una misma red.
    2. b) Enviar datos entre redes diferentes.
    3. c) Proteger la red de amenazas externas.
    4. d) Administrar el suministro de energía en una red.

II- Enunciado del Examen de Excel - Presupuesto con Descuento y IVA 10P

Objetivo:

  • Elabora una plantilla en Excel para calcular presupuestos con la capacidad de aplicar descuentos, calcular el IVA y cumplir con los requisitos de formateo.

Requisitos:

  • Entrada de Datos: 2P
    • Debe haber una sección para ingresar el nombre del cliente, la fecha y los detalles de los productos/servicios.
    • Para cada producto/servicio, se debe ingresar la cantidad y el precio unitario.
  • Cálculos: 5p
    • Calcular el subtotal para cada producto/servicio (cantidad * precio unitario).
    • Aplicar un descuento del 5% sobre el total antes de impuestos.
  • Presentación y Formateo: 3p
    • Utilizar formatos de celda adecuados para la fecha y los números (moneda, separador de miles, etc.).
